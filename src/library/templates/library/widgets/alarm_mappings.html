Help — status → severity mappings

Source is the decoded field to inspect — defaults to status (the wmbusmeters convention). For LoRaWAN, use the JS-codec output field name.

Match is the flag token / value that activates the alarm. It matches as a token within space/comma-separated multi-flag strings, or by equality.

Severity drives the alert level on the consuming instance. Flags not listed here fall back to the instance's built-in driver registry, then to a generic warning.
